CYB1153 | AWS | S3 Static Website

Annuaire des employees

Cette page statique represente la vitrine publique du projet universitaire CYB1153. Elle est hebergee dans un bucket S3 configure en mode static website hosting.

Dans l'architecture cible, l'Application Load Balancer redirige la route /index.html vers ce site statique, tandis que la route /SamplePage.php reste servie par les instances EC2 Apache/PHP.

Verifiez que index.html, error.html et photos.png sont bien presents dans le bucket apres le deploiement Terraform.
